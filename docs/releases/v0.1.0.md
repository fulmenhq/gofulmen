## Quick Start

### Basic Logging

```go
package main

import "github.com/fulmenhq/gofulmen/logging"

func main() {
    logger := logging.NewCLI("myapp", logging.INFO)
    defer logger.Sync()

    logger.Info("Application started")
    logger.WithFields(map[string]any{
        "user": "alice",
    }).Info("User logged in")
}
```

### Configuration Paths

```go
package main

import (
    "fmt"
    "github.com/fulmenhq/gofulmen/config"
)

func main() {
    // Get XDG-compliant config directory
    configDir := config.GetAppConfigDir("myapp")
    fmt.Println("Config directory:", configDir)
    // Output: /Users/user/.config/myapp

    // Get Fulmen-specific paths
    fulmenConfig := config.GetFulmenConfigDir()
    fmt.Println("Fulmen config:", fulmenConfig)
}
```

### Schema Validation

```go
package main

import (
    "github.com/fulmenhq/gofulmen/schema"
)

func main() {
    validator := schema.NewValidator()

    data := []byte(`{"severity": "INFO", "message": "test"}`)
    schemaData := []byte(`{"type": "object", "required": ["message"]}`)

    if err := validator.Validate(schemaData, data); err != nil {
        panic(err)
    }
}
```

## Known Issues

- Schema validation limited to draft 2020-12 (progressive enhancement planned)
- Terminal catalog is static (dynamic detection planned for v0.1.1)
- Logging profiles not yet implemented (SIMPLE → STRUCTURED → ENTERPRISE)
- Foundry module not yet implemented (planned for v0.1.1)
