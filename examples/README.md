# Examples

This directory contains example code demonstrating how to use the Fulmen Go libraries.

## Basic Pathfinder

`basic-pathfinder.go` shows how to:

- Create a Finder instance
- Find files by extension
- Use custom queries with include/exclude patterns
- Handle results

Run with:

```bash
go run basic-pathfinder.go
```

## Running Examples

All examples can be run directly with `go run` since they include a `main` function.

## Learning Path

1. Start with `basic-pathfinder.go` for fundamental usage
2. Explore configuration options
3. Learn about validation and error handling
4. Study the test fixtures for advanced patterns
